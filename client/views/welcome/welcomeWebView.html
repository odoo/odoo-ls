How to install ?

If you see this message, it means that your extension is already installed. But there is some configurations you'll need to do to make it work.
  1. Choose your python version. By default, the extension will run "python3" to execute itself. If you want to use a specific version, you can change the version in the settings. (File > Preferences > Settings > Extensions > Odoo > Python version). You can put a complete filepath or the command name (python3, python3.8, python3.9, etc...). Be sure to use a version greater than 3.8
  2. Install Python dependencies. Use the value you used in the point A (or "python3" if you skipped it) and run the following command: 'your_command' -m pip install parso pygls
  3. The extension is made to work alone, so to avoid double autocompletion and to remove wrong error diagnotics, it could be better to disable your actual Python linter for your workspace. To do that, you can go in the "Extensions" tab, search for your linter and click on "Disable" or "Disable (workspace)".
  4. That's it ! You are ready to use the extension !

How to use the extension ?

The extension work by setting up a configuration, then analysing your code with the loaded configuration.
You have to first create the configurations you'll use later.

Your available configurations are displayed here:

click on it to open the configuration selector panel:

To create a new configuration, click on "Add a configuration". To edit one, click on the wheel next to the configuration you want to edit.

Create your configuration with the information related to the odoo you want to use:

When your configuration is ready, select it for your panel. This selection will run for the current workspace.
While the wheel next to the Odoo button is spinning, the extension is loading the configuration. When it's done, the wheel will stop spinning and the extension will start to help you in your work.
You can use your vscode while the extension is loading, but no feature will be available until the configuration is ready.
